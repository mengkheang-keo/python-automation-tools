# Python Automation Tools

A collection of small Python automation tools built while learning scripting and productivity automation.

## Tools

### Disk Analyzer
Analyzes folders or drives and summarizes disk usage by file extension.

Features:
- Recursive folder scanning
- File extension grouping
- Total size per extension
- Largest file detection
- Human-readable sizes (KB / MB / GB)
