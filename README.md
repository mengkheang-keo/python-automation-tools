# Python Automation Tools

A collection of small Python tools built for learning automation, scripting, and productivity workflows.

These tools focus on automating common tasks such as file system analysis, media downloading, and messaging bots.

---

## Tools

### Disk Analyzer
Analyze folders or drives and summarize disk usage by file extension.

Features:
- Recursive folder scanning
- File type statistics
- Total size per extension
- Largest file detection

📂 Location: [disk-analyzer](./disk-analyzer)

---

### YouTube Video Downloader
Download YouTube videos or playlists using **yt-dlp**.

Features:
- Download single videos
- Download full playlists
- Choose maximum resolution
- Automatic video/audio merging
- Custom filename support

📂 Location: [youtube-video-downloader](./youtube-video-downloader)

---

### Telegram Countdown Bot
A Telegram bot that manages countdown timers and sends reminders.

Features:
- Countdown tracking
- Automated reminder notifications
- Telegram Bot API integration

📂 Location: [telegram-countdown-bot](./telegram-countdown-bot)

---
